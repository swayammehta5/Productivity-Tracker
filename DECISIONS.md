# DECISIONS.md

## Decision 1: Database Selection

Options:

* MongoDB
* PostgreSQL
* MySQL

Selected:
MongoDB

Reason:
Flexible schema and excellent integration with MERN stack.

---

## Decision 2: Authentication

Options:

* Session Authentication
* JWT
* OAuth Only

Selected:
JWT + Google OAuth

Reason:
Provides both traditional and social login support.

---

## Decision 3: Frontend Framework

Options:

* React
* Angular
* Vue

Selected:
React

Reason:
Component-based architecture and strong ecosystem.

---

## Decision 4: Deployment

Options:

* Render
* Railway
* AWS

Selected:
Render

Reason:
Easy deployment and free hosting support.

---

## Decision 5: Styling Framework

Options:

* Bootstrap
* Tailwind CSS
* Material UI

Selected:
Tailwind CSS

Reason:
Fast development and highly customizable UI.
